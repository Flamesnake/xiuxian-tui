# 🌀 修仙模拟器

> 终端 TUI 仪表盘修仙模拟器。在命令行中体验散修的修仙之路。

## 快速开始

```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/Mac: source .venv/bin/activate
pip install -e .
python main.py
```

## 操作方式

| 快捷键 | 操作 | 说明 |
|--------|------|------|
| `C` | 修炼 | 消耗灵气和时间，获得修为 |
| `E` | 探索 | 随机事件，可能触发因果链 |
| `R` | 休息 | 恢复气血和灵气 |
| `D` | 炼丹 | 消耗灵石炼制丹药 |
| `M` | 坊市 | 出售材料获得灵石 |
| `B` | 批量闭关 | 连续修炼10次 |
| `T` | 突破 | 修为满时冲击瓶颈 |
| `P` | 秘境 | 限时秘境探索 |
| `S` | 存档 | 保存当前进度 |

也支持鼠标点击操作按钮。

## 特色

- **资源拮抗**：修为、气血、灵气、寿元四维联动，每个选择都有权衡
- **境界体系**：🟢 炼气期 → 🔵 筑基期 → 🟣 金丹期，大境界突破需渡天劫
- **随机事件**：彩色沉浸文案 + 因果链（灵草移植/妖兽追踪）+ 限时秘境
- **修仙年表**：自动记录修炼历程，回顾成长之路
